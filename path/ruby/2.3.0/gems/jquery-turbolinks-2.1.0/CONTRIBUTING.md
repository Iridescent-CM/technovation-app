# How to contribute

## Run tests

``` sh
# Bundle gems
bundle

# Install npm packages
npm install

# Run tests 
rake test

# ... or run auto tests
bundle exec guard
```

## Sending pull-request

1. Fork
2. Run the tests
3. Add test for your change
4. Make tests green
5. Push
6. Send pull-request

## Please do not

* Driving drunk
* Hide the fact of finding Bigfoot
* Mess with versions, gemspec file, package.json etc
